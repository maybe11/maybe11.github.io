Hexo搭建个人博客并使用github部署

前言 作为一枚程序员有搭建博客的想法很久了，后来由于懒癌发作一直搁置。最近又萌生了个人博客的念头，网上开源的博客系统有很多，一时不知道用哪个好。之所以选用Hexo是正好看到B站羊哥的视频，他在推荐hexo这个框架。Hexo吸引我的主要有两点，第一点Hexo轻便操作简单，仅依赖nodejs就可以安装部署；第二点 Hexo支持github部署，这对没有公网ip的人来说简直就是福音。

https://www.bilibili.com/video/BV1Yb411a7ty

第一章 安装，本地部署

  2. 安装好nodejs后，在命令行输入以下命令就会一键安装hexo框架

  3. 创建一个文件夹，接下来的项目将会构建在这里 我这里创建了hexo-blog目录

  4. 在创建的目录下执行hexo init 脚本会自动构建好hexo项目

  5. 执行hexo s 启动项目 默认的端口是4000 启动完成后访问http://localhost:4000就能看到博客主页了
